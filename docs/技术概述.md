# CampusPilot 技术概述

> 重庆邮电大学个人学业智能助理 — 技术架构简要说明

---

| 层级 | 选型 | 版本 |

## 系统架构

```
用户 (浏览器/飞书)
    │
    ├── Web 界面 (React, localhost:3000)
    │       │ HTTP JSON
    │       ▼
    └── 飞书机器人 (SSH 隧道 → 公网服务器)
            │ HTTPS
            ▼
    ┌─────────────────────────────────────┐
    │        FastAPI (port 8000)           │
    │                                      │
    │  API 路由层 (11 个模块)              │
    │  ├── /api/courses       课表        │
    │  ├── /api/assignments   作业        │
    │  ├── /api/todos         待办        │
    │  ├── /api/rooms         空教室      │
    │  ├── /api/feishu/app    飞书回调    │
    │  ├── /api/config        系统配置    │
    │  ├── /api/llm           AI 对话     │
    │  ├── /api/ai/providers  AI 提供商   │
    │  └── /api/custom-reminders 自定义提醒│
    │                                      │
    │  LLM 模块 (智能分流)                 │
    │  ├── 关键词匹配 → 直接执行工具      │
    │  └── LLM Function Calling →          │
    │       ├── 预定义工具 (12个)         │
    │       ├── Text-to-SQL → SQLite      │
    │       └── 普通对话                   │
    │                                      │
    │  SQLite (data/campus.db)             │
    └─────────────────────────────────────┘
```

## 核心数据模型（9 个）

| 模型 | 表 | 用途 |
|------|-----|------|
| User | users | 用户账号 |
| DataSource | data_sources | 学习通/数你最灵等外部数据源 |
| Course | courses | 课程表（支持周次标记和变地点）|
| Assignment | assignments | 作业（含去重 and 推送状态）|
| Todo | todos | 待办事项（含提醒开关）|
| Notification | notifications | 推送历史记录 |
| SystemConfig | system_configs | 系统配置键值存储 |
| RoomSchedule | room_schedules | 教室课表（查空教室用）|
| CustomReminder | custom_reminders | 自定义定时提醒 |
| AiProvider | ai_providers | 自定义 AI 提供商配置 |

## AI 对话流程

```
用户消息
    │
    ▼
_match_tool() 关键词匹配
    ├── 匹配成功 → 直接执行工具 → 返回结果（百毫秒级）
    │
    └── 未匹配 → chat_with_tools()
                  ├── LLM 决定调用工具 → 执行 → LLM 总结
                  ├── Text-to-SQL → 查库 → LLM 总结
                  └── 普通闲聊 → 直接回复
```

## 定时任务（8 个）

| 任务 | 周期 | 说明 |
|------|------|------|
| sync_courses_daily | 06:00 每日 | 教务课表同步 |
| send_daily_schedule | 07:50 每日 | 课表推送 |
| check_assignment_deadlines | 每小时 | 作业截止提醒 |
| sync_assignments_periodically | 每 30 分钟 | 作业抓取同步 |
| cleanup_old_assignments | 03:00 每日 | 历史作业清理 |
| refresh_room_data_daily | 04:00 每日 | 教室数据刷新 |
| check_custom_reminders | 每 1 分钟 | 自定义提醒 |
| check_todo_reminders | 每 5 分钟 | 待办到期提醒 |

## 爬虫模块（4 个）

| 爬虫 | 目标 | 认证方式 | 网络要求 |
|------|------|----------|----------|
| jwxt_crawler | 教务系统课表 | 无需认证 | 校园网/VPN |
| chaoxing_crawler | 学习通作业 | AES-CBC 加密登录 | 公网 |
| smartestu_crawler | 数你最灵作业 | JWT Token | 公网 |
| room_crawler | 教室课表 | 无需认证 | 校园网/VPN |

## 推送通道（3 个）

| 通道 | 方式 | 用途 |
|------|------|------|
| 飞书 App | HTTP API + SSH 隧道 | 双向对话 |
| 飞书群机器人 | Webhook | 定时推送 |
| Bark iOS | HTTP API | iPhone 通知 |

## 安全设计

- 敏感凭据使用 Fernet 加密存储（非明文）
- Text-to-SQL 仅允许 SELECT，禁止写操作
- 飞书消息 30 秒 TTL 去重
- 前端不存储凭据，全部走后端 API
- 配置支持 `.env` + 数据库双通道

## 环境变量

仅 4 项基础配置存于 `.env`，其余 20+ 项业务配置通过 Web 页面管理并加密入库：

```env
FERNET_KEY=              # 自动生成
TZ=Asia/Shanghai
DATABASE_URL=sqlite+aiosqlite:///data/campus.db
FRONTEND_URL=http://localhost:3000
```
